# NightMode 🌙

> **NightMode** is a high-performance browser extension designed to eliminate eye strain and protect your sleep cycle. By stripping away harsh blue light and providing a granular "Smart Invert" dark mode, it transforms any website into a comfortable, night-vision-friendly experience.

---

## 🚀 Features
- **Triple Filter Modes:** Switch between **Amber** (Blue Light), **Dim** (Brightness reduction), and **Grayscale** (Focus mode).
- **Precision Intensity:** A 0–90% hardware-accelerated slider for perfect comfort in any lighting.
- **Auto-Schedule Engine:** Set your active hours (e.g., 20:00 to 07:00) and let the extension handle the transition automatically.
- **OLED Black UI:** A professional, deep-black interface designed specifically to emit zero blue light while you adjust your settings.
- **Cross-Midnight Logic:** Intelligent background alarms that ensure the filter stays active even if your session crosses into the next day.

---

## 🛠 Tech Stack
- **Vanilla JavaScript:** Clean, dependency-free execution.
- **Chrome Extension API:** Manifest V3 compliant.
- **CSS Hardware Acceleration:** Uses `backdrop-filter` and `mix-blend-mode` for zero layout lag.

---

## 🛠 Support & Updates
- **Manual Updates Only:** This extension does not support automatic updates. Check the repository to ensure you are running the latest version.
- **Refresh Note:** After the initial installation or when updating the schedule, please **refresh your active tabs** to allow the content script to inject correctly.
- **Privacy First:** All scheduling and filter data is stored locally on your machine via `chrome.storage.local`. No data ever leaves your browser.

---

## ⚙️ Installation
1. Download the source code.
2. Navigate to `chrome://extensions/` in your browser.
3. Enable **Developer Mode** in the top-right corner.
4. Click **Load unpacked** and select the folder named **NightMode**.
